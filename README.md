# Chendln

## Don't use this package (not ready).

* It doesn't check end lines, it just changes them, so be aware.

```shell
Usage: 

$ chendln -t lf -i crlf_file -o lf_file
```

```shell
Options:

-t <enc>,                 New endline, lf or crlf.
-i <input>,               Input file.
-o <output>,              Output file (optional).
--Help,                   Show help.
```
